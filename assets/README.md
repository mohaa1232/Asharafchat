# Assets folder

Placeholder folders so `pubspec.yaml`'s asset declarations don't break
`flutter pub get`. Add your real content here:

- `images/` — splash screen image, onboarding illustrations, default avatar
- `icons/` — app launcher icon source (e.g. `app_icon.png`, 1024x1024) for
  use with `flutter_launcher_icons` if you add it later
- `stickers/` — sticker pack images, once you build that feature

Empty folders don't survive in Git/zip by convention, hence the `.gitkeep`
files inside each — safe to delete once you add real assets.
